# CatUni-Angular

### Languages & Frameworks used:
1. HTML
2. CSS
3. TypeScript
4. Angular 5

### To view locally:
1. open Git.
2. clone repository `git clone https://github.com/ShuIly/CatUni-Angular.git`.
4. Run `npm install`.
5. Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
